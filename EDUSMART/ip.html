Materi IPv4

Pengenalan alamat IP versi 4 dalam jaringan komputer

📘 Pengertian IPv4

IPv4 atau Internet Protocol Version 4 adalah sistem pengalamatan yang digunakan untuk memberikan identitas unik kepada setiap perangkat di jaringan komputer.

IPv4 terdiri dari 32-bit yang dibagi menjadi empat oktet dengan format seperti: 192.168.1.1

IP Address memungkinkan perangkat saling berkomunikasi dan bertukar data melalui jaringan.

  • Network ID → menunjukkan alamat jaringan
  • Host ID → menunjukkan perangkat dalam jaringan
  • Public IP → digunakan untuk internet global
  • Private IP → digunakan dalam jaringan lokal

💡 Contoh IPv4

Saat laptop terhubung ke WiFi sekolah, perangkat akan memperoleh IP seperti 192.168.10.5.

📡

Materi Subnetting

Teknik pembagian jaringan komputer

📘 Pengertian Subnetting

Subnetting adalah teknik membagi jaringan besar menjadi beberapa subnet kecil agar jaringan lebih efisien dan mudah dikelola.

Subnetting menggunakan subnet mask untuk menentukan bagian network dan host.

Contoh subnet mask: 255.255.255.0

  • Network ID → alamat awal jaringan
  • Broadcast Address → alamat penyiaran data
  • Host Address → alamat perangkat jaringan

💡 Contoh Subnetting

Subnet mask 255.255.255.0 memiliki kapasitas 254 host.

🔗

Materi Gateway

Penghubung antar jaringan komputer

📘 Pengertian Gateway

Gateway adalah jalur penghubung antara jaringan lokal dengan jaringan lain atau internet.

Gateway biasanya berupa IP router seperti: 192.168.1.1

Semua data menuju internet akan melewati gateway terlebih dahulu.

  • Default Gateway → jalur utama jaringan
  • Router Gateway → penghubung antar jaringan
  • Internet Gateway → akses menuju internet

💡 Contoh Gateway

Jika IP komputer adalah 192.168.1.10, maka gateway biasanya 192.168.1.1.

🚀 Tips Belajar Networking

Latihan konfigurasi jaringan dan subnetting secara rutin akan membantu memahami konsep networking lebih cepat.

  • Pahami konsep dasar IP Address
  • Latihan subnetting setiap hari
  • Gunakan Cisco Packet Tracer
  • Biasakan membaca format CIDR
  • Coba membuat jaringan sederhana sendiri
